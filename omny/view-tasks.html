1.2. User story

At any given time there are many things happening across different aspects of a small business. To keep on top of all these things the user wants a single place to go to see the work to be done.

Warning
 How people manage such work is quite a personal choice as everyone has their own habits and preferences, which is why there are so many tools on the market. Omny’s take is that we should be able to deliver the tasks to the tool of your choice so that the user and for you to use that familiar interface. Therefore we should take care that the existing embedded task view is not over-sold.
